## What is Choccy IDE?

Choccy IDE is a closed-source, full-featured code editor built with Flutter, designed for **Android** and **Windows**. It ships with a built-in Linux environment (proot Ubuntu), an integrated debugger, LSP support, and **Orbit AI** — an autonomous AI coding agent that can design, plan, and execute code on your behalf.

Whether you're coding on your phone during a commute or on your desktop at home, Choccy IDE provides a consistent, professional development experience across platforms.

## Key Features

### Orbit AI Agent
Orbit AI is an autonomous coding partner built directly into the editor. It operates in three modes:

| Mode | Description |
|------|-------------|
| **MINI** | Quick Q&A and simple code edits |
| **MINI+** | File operations and command execution |
| **FULL** | Complete design-to-execution autonomous workflow |

In FULL mode, Orbit can analyze requirements, generate design documents (PRD, architecture, module design), break down tasks, write code, run checkpoints, and auto-fix errors — all autonomously while you stay in control.

### Editor
- Syntax highlighting for 10+ languages
- Auto-completion and code folding
- Multi-cursor editing
- LSP (Language Server Protocol) integration for intelligent completions and go-to-definition

### Built-in Linux Environment
- proot Ubuntu with GCC, Python, Node.js, and more toolchains pre-installed
- Full-featured terminal emulator with multi-tab support

### Debugger
- GDB for C/C++
- debugpy for Python
- Breakpoints, variable inspection, step-through execution

### Project Management
- File browser with tree view
- CMake support
- Project templates for quick start

### Supported Languages

C / C++ · Python · Java · JavaScript · TypeScript · Go · Rust · Lua · C# · Kotlin

## Choccy IDE 是什么？

Choccy IDE 是一款基于 Flutter 构建的闭源全功能代码编辑器，专为 **Android** 和 **Windows** 平台设计。它内置 Linux 环境（proot Ubuntu）、集成调试器、LSP 支持，以及 **Orbit AI** —— 一个能够自主设计、规划和执行代码的 AI 编程智能体。

无论你是在通勤路上用手机写代码，还是在家用电脑开发，Choccy IDE 都能提供一致的、专业的跨平台开发体验。

## 核心功能

### Orbit AI 智能体
Orbit AI 是内置于编辑器中的自主编程伙伴，提供三种工作模式：

| 模式 | 说明 |
|------|------|
| **MINI** | 快速问答和简单代码编辑 |
| **MINI+** | 文件操作和命令执行 |
| **FULL** | 完整的设计到执行自主工作流 |

在 FULL 模式下，Orbit 可以分析需求、生成设计文档（PRD、架构设计、模块设计）、分解任务、编写代码、运行检查点并自动修复错误 —— 全程自主执行，同时你掌控全局。

### 编辑器
- 支持 10+ 种语言的语法高亮
- 代码补全与代码折叠
- 多光标编辑
- LSP（语言服务器协议）集成，智能补全与跳转定义

### 内置 Linux 环境
- proot Ubuntu，预装 GCC、Python、Node.js 等工具链
- 功能完整的终端模拟器，支持多标签页

### 调试器
- GDB 调试 C/C++
- debugpy 调试 Python
- 断点设置、变量查看、单步执行

### 项目管理
- 文件浏览器（树形视图）
- CMake 支持
- 项目模板快速创建

### 支持语言

C / C++ · Python · Java · JavaScript · TypeScript · Go · Rust · Lua · C# · Kotlin
