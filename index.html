✨ 核心功能

桌面级的开发体验，移动端的便携优势

📝

智能代码编辑

语法高亮、代码补全、括号匹配、多光标编辑、代码折叠，提供流畅的编辑体验。

🔧

完整编译环境

内置 proot Ubuntu 环境，支持 GCC、G++、CMake、Python、Java、Node.js 等工具链。

🐛

调试支持

支持 GDB 调试 C/C++，debugpy 调试 Python。设置断点、单步执行、查看变量值。

📁

项目管理

完整的文件浏览器，支持 CMake 项目、多文件编辑、项目模板快速创建新项目。

🖥️

集成终端

内置终端模拟器，支持多标签页，可直接运行 Linux 命令，安装软件包。

🌐

LSP 智能补全

支持语言服务器协议，提供智能补全、跳转定义、查找引用、重命名等高级功能。
